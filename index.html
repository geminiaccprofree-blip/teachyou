Functional Components

What Every Mode
Ships With

Shared intelligence layer across all 4 modes — every output is pedagogically complete by default.

Analogy Generator

Complex concepts mapped to familiar real-world parallels. LLM identifies the closest structural analogy from the learner's declared background.

Step-by-Step Explainer

Every concept broken into numbered atomic steps. No assumed knowledge. Each step self-contained and reversible for targeted review.

Common Errors Module

AI catalogs the 5 most frequent mistakes learners make on each topic, with before/after corrected examples for each error pattern.

Practice Task Generator

Generates 3 graded practice tasks per topic — beginner, intermediate, and applied. Each includes success criteria and a model answer.

Flashcard Export

Auto-exports Anki-compatible .apkg decks. Front: concept. Back: full explanation + example + common error note. Sorted by spaced-repetition priority.

Mastery Dashboard

Visual progress tracking per module. Chart.js radar chart shows concept coverage. Session analytics stored in LocalStorage for offline access.

PDF Export Engine

Full course summary, quiz results, and certificate exportable as branded PDF via html2canvas + jsPDF. No server needed — runs entirely client-side.

Bilingual Output

All content generated in English + Urdu side-by-side for Pakistani learners. Toggle per-module. Space Nastaleeq font loaded via Google Fonts for Urdu rendering.

Technology

Stack Powering
SYNAPSE

Every tool chosen for maximum output quality at zero cost — built on open APIs, client-side rendering, and orchestration-first architecture.

Claude / GPT-4

Core LLM backbone — handles Pareto extraction, Socratic prompting, analogy generation, script writing, and all structured content output across all 4 modes.

n8n Orchestration

Automated Pipeline mode triggers: topic in → script + quiz + flashcards + slides + email sequence out. Self-hosted, zero recurring cost, full workflow visibility.

D-ID Avatars

Auto-generates AI avatar lecture videos from the script output. Supports 100+ languages. Used in the Automated Pipeline to produce video lectures without a camera.

Suno AI

Mnemonic song generation in Unconventional Mode. LLM writes the lyrics (formula + hook), Suno renders a 30-second audio track. Students hum their way to mastery.

Supabase

Leaderboard, user progress, quiz scores, and streak data stored in Supabase Postgres. Avoids PostgreSQL reserved words — sort_order, record_id naming convention enforced.

Phaser.js

Browser-based game engine powering Boss Fight Mode. Each topic sub-concept is a mini-boss. Sprites, health bars, and XP all rendered in-browser — no install needed.

html2canvas + jsPDF

Client-side PDF generation for certificates, cheat sheets, and course summaries. No server. Renders any HTML section to canvas, then exports as a print-ready PDF.

Chart.js + D3.js

Chart.js for mastery dashboards and progress rings. D3.js for interactive mind maps in 80/20 Mode — force-directed graph shows concept relationships at a glance.
